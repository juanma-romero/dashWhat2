# DashWhat

web app dashborad alternativo a Whatsapp web

***Tecnologias aplicadas:***

* ci/cd: github
* front: react, tailwind, vite.js
* back: express 
	(front y back con websocket)	
* api: baileys
* comunication: redis
* db: postrgresSQL
* seguridad: token JWT
* log: Sentry
* asistiendo en todos lados: OpenAI gpt-4o y Gemini Code Assist
